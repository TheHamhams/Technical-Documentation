Introduction

There are many ways to end a game in unity:

  • when the player finishes a level
  • when their health gets to zero
  • when they drive off a cliff

In this documentation we will go over one way to create a game timer. This is by no means the only way to do this, but this is what I have done in multiple games and it works well.

What you should already know

For this guide to makes sense you should have knowledge of the following subjects:

  • Basic knowledge of the Unity game engine
  • Basic knowledge of C#
  • How to make if statements
  • How to create methods
  • How InvokeRepeating() works
Declaring Variables

You dont need to name your variables exactly the same, just make sure they are memorable and make sense so you or anyone else who looks/works with the code can easily identify it

int timeLeft; bool isActive; TextMeshProUGUI timerText;

These are used to store the following:

  • timeLeft is used to store the length of your timer.
  • isActive is how I store whether or not the game is active or not.
  • timerText is a TextMeshPro UI object that I use to display the time on the game screen. You don't have to use TMPro, but I like it better than the default.
Timer method

Next we need to create a method to contain the timer operation. This is how I set it up:

void Timer() { if (timerActive) { if (timeLeft > 0) { timeLeft--; timerText.text ="Time left: " + timeLeft; } if (timeLeft == 0) { timerLeft = 0; isActive = false; timerText.text ="Time left: " + timeLeft; //here is where you put the method for ending the game, I usually name it GameOver() } } }

So what does it all mean?

  • if (timerActive)

    This makes it so the timer only decreases if the game is active

  • if (timeLeft > 0) { timeLeft--; timerText.text ="Time left: " + timeLeft;

    So long as the timer is more than zero it will decrease by one every time it is called. It also also updates the UI to show the current ammonut of time left.

  • if (timeLeft == 0) { timeLeft = 0; isActice = false; timerText.text ="Time left: " + timeLeft; // GameOver() }

    This stops the game once the timer gets to zero> you should also add a method for that the program should do once the game ends which I usually name GameOver().

Starting Method

Now in order to make this work you need to instantiate this method. You can also do this multiple ways, but here is a simple version I use.

void StartGame() { isActive = true; timeLeft = 60; InvokeRepeating("Timer", 1, 1); }

This is a pretty straight forward method that activates the game, creates a length on the timer, and activates the Timer method every second.

You have done it! that is all you need to do to make a simple timer for your game.
